# مواقيت الصلاة

مواقيت هو برنامج تفاعلي لسطر الأوامر لحساب مواقيت الصلاة باستخدام Node.js و Inquirer. يحسب البرنامج مواقيت الصلاة اليومية للمدن حول العالم مع إمكانية اختيار طريقة الحساب المناسبة.

## المميزات

- البحث في أكثر من 234,000 موقع من قاعدة بيانات GeoNames للمدن.
- اقتراح فرق التوقيت الأساسي UTC تلقائيا حسب المدينة المختارة والتاريخ.
- السؤال عن تطبيق التوقيت الصيفي أو عدم تطبيقه.
- تذكر آخر اختيارات المستخدم، مثل الموقع، طريقة الحساب، طريقة العصر، فرق التوقيت، إعداد التوقيت الصيفي، والارتفاع.
- دعم طرق حساب شائعة:
  - الهيئة المصرية العامة للمساحة
  - رابطة العالم الإسلامي
  - أمريكا الشمالية (ISNA)
  - جامعة العلوم الإسلامية في كراتشي
  - أم القرى، مكة
  - دبي
  - قطر
  - الكويت
  - لجنة رؤية الهلال
  - سنغافورة
  - تركيا / ديانت
  - طهران
  - الإعداد الأصلي للبرنامج
  - زوايا مخصصة للفجر والعشاء
- دعم طريقة العصر القياسية وطريقة العصر الحنفي.

## المتطلبات

- Node.js إصدار 20 أو أحدث

## التثبيت

ثبت الحزم المطلوبة:

```bash
npm install
```

## الاستخدام

شغل البرنامج:

```bash
npm start
```

بعد ذلك اختر الموقع، التاريخ، فرق التوقيت الأساسي UTC، إعداد التوقيت الصيفي، طريقة حساب مواقيت الصلاة، طريقة العصر، والارتفاع عن سطح البحر.

## تذكر الاختيارات

يحفظ البرنامج آخر اختياراتك في مجلد إعدادات المستخدم:

- Windows: `%APPDATA%\\mawakit\\preferences.json`
- macOS/Linux: `$XDG_CONFIG_HOME/mawakit/preferences.json` أو `~/.config/mawakit/preferences.json`

عند التشغيل التالي سيعرض البرنامج الموقع السابق ويستخدم آخر الإعدادات كقيم افتراضية.

## مصدر البيانات

بيانات المواقع مولدة من ملف GeoNames `cities500`، مع تنسيق أسماء الدول والمناطق من ملفات GeoNames المساعدة.

بيانات GeoNames مرخصة برخصة Creative Commons Attribution 4.0:
https://creativecommons.org/licenses/by/4.0/

المصدر:
https://download.geonames.org/export/dump/

## التطوير

تشغيل الاختبارات:

```bash
npm test
```

فحص صياغة الملفات:

```bash
node --check ./src/cli.js
node --check ./src/prayer-times.js
node --check ./scripts/import-geonames.js
```

إعادة توليد بيانات المواقع بعد تنزيل ملفات GeoNames داخل `tmp/geonames`:

```bash
node ./scripts/import-geonames.js
```

## ملاحظات

قواعد المناطق ذات خطوط العرض العالية غير مطبقة حتى الآن. إذا لم يكن ممكنا حساب الشروق أو الغروب أو الفجر أو العشاء فلكيا في موقع أو تاريخ معين، قد تظهر النتيجة `Unavailable`.
